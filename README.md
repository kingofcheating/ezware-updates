## Features

- Visual overlays and aim tools
- Profiles and privacy mode
- Automatic frame pacing and updates
- Read-only mode by default

## Controls

| Key | Action |
| :--- | :--- |
| `Insert` | Toggle menu |
| `F10` | Privacy mode |
| `End` | Safe exit |
